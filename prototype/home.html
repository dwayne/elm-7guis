7GUIs

by Dwayne Crooks

===================================================================================================================================================================

7GUIs (7 Graphical User Interfaces) is a programming benchmark to compare implementations in terms of their notation. 7GUIs defines 7 tasks that represent typical challenges in GUI programming. This site implements the 7 tasks using HTML, CSS & ELM. Click a link below to see a description of a task, the rendered GUI & links to its code.

  2. Temperature Converter
  3. Flight Booker
  4. Timer
